创建文章

POST /v1/community/posts/create

请求头:

名称必须备注
token是无

请求体:

{  
  "baId": 40, // 文章分区ID
  "groupId": "123", // 文本引用群聊ID
  "title": "测试文章标题", // 文章标题
  "content": "测试文章内容", // 文章内容
  "contentType": 1 // 文章内容类别，1-文本，2-markdown
}

响应体:

{
  "code": 1, // 请求状态码，1为正常
  "data": {
    "audioUrl": 123, // 文章ID
  },
  "msg": "success" // 返回消息
}
