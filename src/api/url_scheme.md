## 添加聊天对象

yunhu://chat-add

参数:

| 名称 | 必须 | 备注                                                    |
| ---- | ---- | ------------------------------------------------------- |
| id   | 是   | 要添加的目标对象ID                                      |
| type | 是   | 要添加的目标对象类别，用户-user，群聊-group，机器人-bot |

实例（添加ID为123的用户）：

```text
yunhu://chat-add?id=123&type=user
```

## 跳转指定文章

yunhu://post-detail

参数:

| 名称 | 必须 | 备注               |
| ---- | ---- | ------------------ |
| id   | 是   | 要跳转的目标文章ID |

实例（跳转到ID为111的文章）：

```text
yunhu://post-detail?id=111
```

## 跳转指定文章分区

yunhu://alley-detail

参数:

| 名称 | 必须 | 备注                   |
| ---- | ---- | ---------------------- |
| id   | 是   | 要跳转的目标文章分区ID |

实例（跳转到ID为41的文章分区）：

```text
yunhu://post-detail?id=41
```

## 观看广告

yunhu://ad

参数:

| 名称 | 必须 | 备注           |
| ---- | ---- | -------------- |
| id   | 是   | 要观看的广告ID |

实例（观看广告ID为123的广告）：

```text
yunhu://ad?id=123
```
